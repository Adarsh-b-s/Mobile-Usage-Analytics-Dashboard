
# 📱 Smartphone Usage Analytics & Insight Dashboard

## 📌 Project Overview
This Power BI dashboard analyzes smartphone usage patterns, screen time habits, addiction levels, productivity, stress levels, and overall digital wellbeing of users.

The dashboard provides meaningful insights using interactive visualizations and KPI cards.
